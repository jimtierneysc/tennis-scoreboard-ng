animateChange

Description

Methods to animate some elements in a view by controlling a sequence of delays and state changes.

Methods

  • hideThenShow(change, hideAndShow)

    Given a change function and some other callbacks, animate the elements before and after making a change.

    Parameters
    ParamTypeDetails
    changeFunction

    function to make the change

    hideAndShowObject

    Contains Function members: hideChanging, hideChanged, showChanged and reset.

  • promiseHideThenShow(promise, change, hideAndShow, reject, final)

    Given a promise and some callbacks, animate the elements before and after making a change.

    Parameters
    ParamTypeDetails
    promiseObject

    A promise to complete before make a change

    changeFunction

    Function to make the change

    hideAndShowObject

    Contains Function members: hideChanging, hideChanged, showChanged and reset.

    rejectFunction

    Function called when promise is rejected

    finalFunction

    Function called when promise is finalized

  • toggleShow(toggle, enableNgClass, reset)

    Given a toggle function and two other callbacks, animate the elements that are to be shown or hidden by toggling a setting.

    Parameters
    ParamTypeDetails
    toggleFunction

    Function to toggle a setting

    enableNgClassFunction

    Function to set flags to enable animation

    resetFunction

    Function to set flags to disable animation
