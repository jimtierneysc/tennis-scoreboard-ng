animateChange#promiseHideThenShow

Description

Given a promise and some callbacks, animate the elements before and after making a change.

Usage

new animateChange#promiseHideThenShow(promise, change, hideAndShow, reject, final);

Parameters

ParamTypeDetails
promiseObject

A promise to complete before make a change

changeFunction

Function to make the change

hideAndShowObject

Contains Function members: hideChanging, hideChanged, showChanged and reset.

rejectFunction

Function called when promise is rejected

finalFunction

Function called when promise is finalized
