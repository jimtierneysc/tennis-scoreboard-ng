errorsMapper

Description

Given a hash, return a new hash that groups values together. Useful for processing the errors generated by Rails.

Usage

errorsMapper(data, names, map, defaultKey);

Parameters

ParamTypeDetails
dataObject

Input hash. Values are expected to be strings or string arrays.

namesArray

Key names to copy.

mapObject

Map original key names to new key names.

defaultKeyString

Key name to use for all of the errors that don't match a known key name (in names param). Each value will be prefixed with the original key name.
